Questions fréquentes

Voici quelques unes des questions fréquemment posées sur Libreosteo.

Faut-il une connexion internet pour que cela fonctionne ?

Non, pas du tout. Libreosteo fonctionne autonome installé sur votre machine et n'a besoin d'aucune connexion internet pour fonctionner. Toutefois, si vous êtes partagés entre deux cabinets distants, une solution en ligne est possible en installant Libreosteo sur un serveur (comme pour la démonstration disponible sur ce site), et une connexion sécurisée à ce serveur. Contactez-nous pour plus d'informations sur ce sujet.

Peut-on travailler à plusieurs sur le logiciel ?

Oui vous pouvez. Et sans connexion internet. Installez Libreosteo sur une machine (ordinateur ou mac) branchée sur votre réseau local (wifi ou câble) et accédez à l'instance Libreosteo à travers le réseau local, il est non accessible depuis l'extérieur. Pour celà, il vous faut identifier l'adresse IP sur le réseau local de la machine sur laquelle est installée Libreosteo.
Pour Windows, ces pages ( http://windows.microsoft.com/fr-fr/windows/find-computers-ip-address#1TC=windows-7 ou http://assistance.orange.fr/ordinateurs-peripheriques/installer-et-utiliser/la-configuration-d-un-reseau/windows-8/adresse-ip-l-identifier-sous-windows-8_26297-26944 ) peuvent vous aider à trouver cette adresse. Il vous faut ensuite adjoindre le port 8085 à l'adresse ( par exemple : http://192.168.0.15:8085/ )
Pour MacOS, il vous faut ouvrir vos "Préférences Systèmes" puis "Réseau" pour obtenir votre adresse ip sur le réseau connecté. Ensuite, il vous faut adjoindre le port 8085 à l'adresse obtenue ( par exemple : http://192.168.0.15:8085/ ). Sur les autres machines connectées à ce réseau, il suffit de saisir cette adresse dans le navigateur pour y accéder.

Sur quel équipement je peux l'installer ?

Libreosteo ne peut être installé que sur des ordinateurs. Les tablettes et smartphones ne peuvent héberger Libreosteo. En revanche, vous pouvez y accéder depuis ces équipements s'ils sont connectés au wifi de votre réseau local et l'ordinateur qui héberge Libreosteo y est connecté aussi. Reportez vous à la question précédente pour cela.
Libreosteo peut être installé sur des environnements Windows (XP, 2000, Vista, Seven, Windows 8 et Windows 10) et Mac OS X (10.6 - Snow Leopard et supérieure).
Il est possible évidemment d'exécuter Libreosteo sur Linux en suivant ces indications : README. Mais vous pouvez aussi envisager de monter Libreosteo sur un cloud Linux et multiplier les instances. Le pouvoir est à vous. Pour plus d'informations sur ce sujet, consultez notre wiki sur Docker

J'ai une erreur d'authentification du développeur à l'installation, comment faire ?

Cette erreur provient surtout sur les environnements MacOS X. Depuis quelques versions, Apple oblige les développeurs à s'acquitter d'une licence pour supprimer ce message. La philosophie de Libreosteo n'est pas de s'enfermer dans des principes d'acquisition pour avoir accès. Vous êtes libre d'accéder à Libreosteo et d'installer Libreosteo sur votre machine. Cette pratique est une pratique financière pour inciter les entreprises qui diffusent leur logiciel sur Mac OS X à payer des droits à Apple. Vous verrez donc ce message à chaque installation ou mise à jour de Libreosteo. Pour pouvoir installer Libreosteo, malgré cette alerte je vous invite à suivre la documentation Apple sur ce sujet : Gatekeeper

Ma base patient est-elle sauvegardée sur internet ? Y a-t-il une fuite d'information ? Comment faire en cas de crash de mon disque ?

Aucune connexion internet n'est nécessaire, la base patient est stockée sur votre ordinateur où est installé Libreosteo. Cela signifie que vous avez la responsabilité de cette base patient. Cela garantit la confidentialité des informations stockées, et leur intégrité. En revanche, aucune sauvegarde sur internet n'est proposée à ce jour gratuitement. Vous pouvez monter des solutions de sauvegarde en ligne pour y palier avec Hubic (la sauvegarde restera en France) ou Dropbox (la sauvegarde peut être aux USA ou en Asie).
La base patient est stockée dans le fichier db.sqlite3 qui est stocké à côté de votre logiciel sous Microsoft Windows ou bien dans Library/Application Support/Libreosteo pour les utilisateurs Mac OS X. Sauvegardez ce fichier sur un disque externe est aussi possible. En cas de crash, réinstallez Libreosteo, et déposez le fichier au même endroit. Vous retrouverez toutes vos données.

J'utilise un autre logiciel actuellement, puis-je importer mes données dans Libreosteo ?

Libreosteo ne fournit aucun plugin d'import de base actuellement. En revanche, une API Rest est disponible pour importer les informations. Des scripts de transfert de données peuvent être également utilisés en utilisant le shell de Django. Contactez le développement si vous souhaitez développer un plugin d'import/export.

C'est gratuit ? Mais comment est-ce possible ?

Libreosteo est libre. Le développeur a fait le choix actuellement de le fournir gratuitement pour permettre aux jeunes ostéopathes nouvellement installés de bénéficier d'un produit simple et fiable pour démarrer leur cabinet. Vous êtes libres ensuite de le quitter de choisir un autre logiciel plus tard. Libreosteo est animé par des valeurs d'échange, de collaboration et les contributions sont importantes. Le développeur vous offre Libreosteo. En échange, apportez-lui votre soutien, aidez le à l'améliorer, acceptez les erreurs, et participez à la communauté des utilisateurs pour aider les autres à l'utiliser ou à démarrer. Afin de pérenniser son développement et pouvoir couvrir plus de fonctionnalité, vous pouvez effectuer un don.

J'ai perdu mon mot de passe, comment puis-je faire ?

Malheureusement, le mot de passe est chiffré et vous ne pourrez pas le récupérer. Cela va donc être difficile de récupérer le mot de passe Administrateur. En revanche, depuis l'interface de gestion du cabinet, vous pouvez changer le mot de passe d'un des collaborateurs du cabinet. Pratique s'il l'a perdu, ou bien dans le cas où il doit quitter votre cabinet.
A l'extême limite, si vous êtes totalement bloqué, vous pouvez changer le mot de passe administrateur de Libreosteo en effectuant les étapes suivantes :

  • Sous Windows :
    • Se positionner à l'aide du navigateur de fichier dans le répertoire d'installation de Libreosteo (ex : C:\Program Files(x86)\Libreosteo)
    • Menu Fichier > Ouvrir Windows PowerShell > Ouvrir Windows PowerShell en tant qu'administrateur
    • taper : 
      ./manager.exe changepassword nom_utilisateur
      où 
      nom_utilisateur
      est votre nom d'utilisateur sous Libreosteo
  • Sous Mac OS X:
    • Ouvrir un Terminal
    • Taper : 
      cd /Applications/Libreosteo.app/Contents/MacOS
    • puis taper : 
      ./manage changepassword nom_utilisateur
      où 
      nom_utilisateur
      est votre nom d'utilisateur sous Libreosteo
